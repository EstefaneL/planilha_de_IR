# planilha_de_IR

Planilha para Declaração de Imposto de Renda

Projeto desenvolvido como parte do bootcamp da DIO (Digital Innovation One) com o objetivo de criar uma planilha organizada e funcional para auxiliar no controle de informações necessárias para a declaração de Imposto de Renda.

Sobre o Projeto

A planilha LION MODE foi criada para centralizar e estruturar os principais dados exigidos na declaração anual de Imposto de Renda Pessoa Física (IRPF), facilitando o preenchimento e a organização das informações financeiras.

O modelo é dividido em três seções principais:


1. Dados do Titular

Área destinada ao preenchimento de informações pessoais do contribuinte, como:

Nome completo; CPF; Data de nascimento; Título de eleitor; Endereço completo; Contatos (telefone e e-mail); Informações adicionais (dependentes, residente no exterior, etc.)

Essa seção concentra os dados cadastrais básicos necessários para a declaração.


2. Informes de Rendimentos Bancários

Espaço para registrar os valores declarados por cada instituição financeira.

Funcionalidades:

Cadastro de múltiplos bancos; Identificação do banco (código e nome); Valor atual informado no informe de rendimentos; Campo para anexar o comprovante (PDF); Cálculo automático do valor total consolidado

Essa aba facilita a visualização global dos rendimentos financeiros do contribuinte.


3. Notas Bancárias / Extratos de Holerites

Seção destinada ao controle mensal das entradas de receita.

Campos disponíveis: Data; Categoria (ex: Holerite); Valor

Permite registrar rendimentos mês a mês para melhor organização e conferência dos valores declarados.


Tecnologias Utilizadas

Microsoft Excel
